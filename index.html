MeltOS Project

The MeltOS Project is a Python 3 script that acts as a "mockery of an OS", of sorts. The goal is to make this little Python script into a REAL OS one day.

FAQ

Q: What is "get"?
A: "get" is the "package manager" of MeltOS. It allows for easy install and integration of Python or moexe files.

Q: What's an "moexe" file?
A: An "moexe" file is a MeltOS executable. It can be ran using the "scr" command, or the filename if the file is integrated.

Q: Why was MeltOS made?
A: One day, I (melt) was bored, and decided it'd be fun to make an OS, that's not an OS.
